== Description == 
 
Software shop is a plugin that allows you to integrate thousands of shareware products handled and distributed by RegNow.com into your Wordpress site. 
If your visitors will buy software from your site you will earn commission (up to 75%). 
Just install plugin, register yourself as a Regnow affiliate here: https://www.regnow.com/signup/affiliate and that's all.
This plugin uses its own permalinks and ads necessary meta tags so its SEO friendly.

Also... Don't forget to add your affiliate ID into "settings".
 
 
== Installation == 

To install: 
 

1. Drop the 'software-shop' folder into your 'wp-content/plugins' folder 
 
2. Go to your Administration: Plugins page and activate the "Software Shop" plugin

3. Go to Administration: Settings: Software Shop and configure your plugin
 
 
To configure: 
 
1. Go to Settings / Software Shop 
2. Register yourself as a Regnow affiliate here: https://www.regnow.com/signup/affiliate
3. Enter your affiliate ID into text box and save changes. 
4. Enjoy
